# C Practical Assignments

This repository contains a complete collection of C programming practical assignments created as part of academic coursework.

The programs cover fundamental to intermediate concepts of the C programming language, including data types, operators, conditional statements, loops, functions, pointers, and pattern printing.

Each assignment is organized into separate folders with properly named source files to maintain clarity and readability.

## Topics Covered
- Data types and size of variables
- Arithmetic, relational, and logical operators
- Conditional statements and loops
- Salary calculation and authentication programs
- Star and pattern printing
- User-defined functions
- Call by reference
- Pointer-based operations

## Repository Structure
Each assignment is stored in its own folder for better organization and easy evaluation.
